Deep Remote Control

Personal Research Project

Project

This project seeks to answer the question as to whether controlling an aircraft from 3rd-person perspective is a viable solution for GPS-denied situations. It uses deep learning to estimate the orientation of an aircraft and a point cloud with a simple flight model to get the position of the aircraft. In this way, deep learning becomes a new sensor that could potentially replace GPS on an aircraft.
